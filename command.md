
## 通信命令(波特率9600)

` 减速电机 `
| 命令  | 动作  |
|-----|-----|
| 'w' | 前进 |
| 'q' | 停止 |
| 's' | 后退 |
| 'a' | 左转 |
| 'd' | 右转 |

` LCD显示 眼球 `
| 命令  | 动作  |
|-----|-----|
| 'j' | 向左看 |
| 'l' | 向右看 |
| 'i' | 失望表情 |
| 'k' | 自然表情 |

` 头部动作 `
| 命令  | 动作  |
|-----|-----|
| 'f' | 抬头 |
| 'g' | 头部向前 |
| 'g' | 头部向后 |

` 手臂动作 `

| 命令  | 动作  |
|-----|-----|
| 'b' | 左手臂低 右手臂高 |
| 'n' | 自然状态 |
| 'm' | 左手臂高 右手臂低 |
